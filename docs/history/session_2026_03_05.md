L'option `-a` demande à Git d'ajouter automatiquement tous les fichiers modifiés et supprimés. Sur Windows, si un fichier est temporairement inaccessible (verrouillé par l'antivirus ou l'IDE), Git peut interpréter cela comme une suppression définitive dans l'index.
